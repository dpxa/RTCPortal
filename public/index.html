About

RTCPortal is a file-sharing tool that enables direct file transfers using WebRTC. Unlike cloud storage services, it doesn't store files on a central server. Files are sent through a private peer-to-peer connection.

A signaling server using Socket.io helps users connect but is only involved in the initial handshake â€“ it does not store or handle file transfers. Once connected, files are exchanged directly.
